Frappe Learning is an easy-to-use, fully open source Learning Management System that helps instructors and organizations bring structure to their educational content. Built on the Frappe Framework with a modern Vue.js frontend, it gives you everything you need to create courses, manage learners, run live classes, and issue certifications — all in one place.

## Get started

<Steps>
  <Step title="Deploy Frappe Learning">
    Use [Frappe Cloud managed hosting](/installation/managed-hosting) for a one-click setup, or [self-host](/installation/self-hosting) on your own server in about 5 minutes.
  </Step>
  <Step title="Create your first course">
    Log in as an instructor, navigate to **Courses**, and click **New Course**. Add chapters and lessons to organize your content.
  </Step>
  <Step title="Enroll learners">
    Share your course link publicly or create a batch to group learners and manage enrollment manually.
  </Step>
  <Step title="Assess and certify">
    Add quizzes or assignments to lessons and grant certificates when learners complete the course.
  </Step>
</Steps>

## Built on open source

Frappe Learning is licensed under AGPL-3.0 and built on two open source foundations:

- **[Frappe Framework](https://github.com/frappe/frappe)** — a full-stack Python web application framework that powers the backend, database, and REST API layer.
- **[Frappe UI](https://github.com/frappe/frappe-ui)** — a Vue-based component library providing the modern, responsive user interface.

<Note>
  Frappe Learning is self-hosted software. You own your data. There is no vendor lock-in.
</Note>
