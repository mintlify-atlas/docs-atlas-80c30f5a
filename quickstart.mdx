---
title: "Get started with Frappe Learning"
description: "Set up Frappe Learning using Docker in minutes. This guide walks you through launching a local instance and creating your first course."
---

This page walks you through the fastest way to get Frappe Learning running on your machine. The Docker setup requires only Docker and docker-compose — no manual dependency installation needed. For a production deployment, see [managed hosting](/installation/managed-hosting) or [self-hosting](/installation/self-hosting).

<Steps>
  <Step title="Create a working directory and download the required files">
    Create a folder for the project, then download the Docker Compose file and the initialization script:

    ```bash
    mkdir frappe-learning
    cd frappe-learning

    # Download the Docker Compose file
    wget -O docker-compose.yml https://raw.githubusercontent.com/frappe/lms/develop/docker/docker-compose.yml

    # Download the setup script
    wget -O init.sh https://raw.githubusercontent.com/frappe/lms/develop/docker/init.sh
    ```

    The Compose file starts three services: MariaDB, Redis, and the Frappe bench container. The `init.sh` script initializes the bench, installs the LMS app, and creates the site on first run.
  </Step>

  <Step title="Start the containers">
    Run the following command from inside the `frappe-learning` directory:

    ```bash
    docker compose up -d
    ```

    The `-d` flag runs the containers in the background. The first run takes a few minutes while the bench is initialized and the LMS app is installed.
  </Step>

  <Step title="Open Frappe Learning in your browser">
    Once the containers are running, open the following URL:

    ```
    http://lms.localhost:8000/lms
    ```

    Log in with the default credentials:

    | Field    | Value         |
    | -------- | ------------- |
    | Username | Administrator |
    | Password | admin         |
  </Step>
</Steps>

<Tip>
  Now that you are logged in, head to **Courses** and click **New Course** to create your first course. You can add chapters and lessons to organize your content, then share the course link with learners.
</Tip>
