---

Frappe Cloud is the official managed hosting platform for Frappe applications. It handles installation, setup, upgrades, monitoring, maintenance, and support for your deployment, so you can focus on building and delivering courses rather than managing infrastructure.

<Steps>
  <Step title="Sign up for Frappe Cloud">
    Go to [frappecloud.com/lms/signup](https://frappecloud.com/lms/signup) and create an account. The sign-up page is pre-configured for deploying Frappe Learning.
  </Step>

  <Step title="Follow the setup wizard">
    After signing up, the Frappe Cloud setup wizard guides you through choosing a plan, selecting a region, and naming your site. Frappe Cloud provisions the server and installs Frappe Learning automatically.
  </Step>

  <Step title="Access your site">
    Once provisioning is complete, Frappe Cloud provides you with a site URL. Open it in your browser and log in with the administrator credentials created during setup.
  </Step>
</Steps>

## What Frappe Cloud manages for you

Frappe Cloud is a fully featured developer platform that takes care of the following for every hosted site:

- **Installation and setup** — Frappe Framework, the LMS app, and all dependencies are installed and configured for you.
- **Upgrades** — new releases of Frappe Learning can be applied from the Frappe Cloud dashboard with a single click.
- **Monitoring** — your site is monitored around the clock with automated alerting.
- **Maintenance and support** — the Frappe Cloud team provides infrastructure support so your site stays available.

<Note>
  Frappe Cloud is [open source](https://github.com/frappe/press). You can also self-host the Frappe Cloud platform itself if you prefer full control over the management layer.
</Note>
